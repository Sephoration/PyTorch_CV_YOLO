期中考核报告填写内容
项目：基于车辆轨迹与角度变化的疑似变道行为检测
====================================================

==================== 一、项目研究目标 ====================

本项目旨在利用车辆轨迹与方向角度变化，从俯视交通视频中检测疑似变道行为。
通过YOLO目标检测与追踪提取车辆运动轨迹，计算连续轨迹点之间的方向向量与
角度差序列，基于双窗口角度变化判定、净角度变化检查和横向位移三重确认机制，
判断车辆是否存在变道嫌疑。
之所以称为"疑似变道"，是因为本方法仅基于视觉轨迹分析识别与变道特征相符
的车辆——车辆的自然转向、弯道行驶等也可能触发检测，故所有检测结果均为
"疑似"而非绝对确认。

==================== 二、视频来源与选择 ====================

[内容1]
本项目选用俯视（高空/天桥）视角拍摄的交通监控视频作为数据源。视频场景包含
从左到右行驶的主干道及右侧延伸至画面顶部中央的弯道。俯视角度可清晰呈现
车辆运动轨迹，避免路面视角常见的遮挡问题，利于持续追踪每辆车的完整路径。

[内容2]
视频为白天正常光照条件下拍摄，无特殊光照处理。车辆在画面中呈现为可追踪的
连续运动目标，车辆之间遮挡较少，适合基于轨迹的变道分析。画面分辨率适中，
车辆像素大小随距离变化，远处车辆位移量较小是本算法面临的主要挑战之一。

==================== 三、算法说明 ====================

本算法基于YOLO（Ultralytics）目标检测与追踪框架，为每辆车分配唯一的Track ID
并记录其平滑轨迹（使用指数平滑减少噪声）。每隔ANGLE_STEP帧取两个轨迹点
计算方向向量(vx, vy)，通过atan2(vy, vx)得到行驶方向角度。将连续角度值做
差分得到角度差序列，利用双滑窗策略进行判定：

(1) 短窗口（8帧）检测快速变道产生的角度突变
(2) 长窗口（20帧）检测缓慢累积的变道趋势

再通过净角度变化检查排除来回摆动的噪声，结合轨迹横向位移确认最终判定。
同时加入最低速度过滤（小于2.5像素/帧的静止车辆跳过分析），避免静止车辆
角度噪声导致的误报。

==================== 四、参数调整过程 ====================

初始使用路侧视角的默认参数，在俯视画面上出现大量误报和漏报。逐步调整过程：

① 新增速度过滤：MIN_SPEED = 2.5像素/帧
   消除静止车辆（ID 6, 7, 9）角度噪声导致的误报

② 横向位移轴由'x'改为'y'
   主车道为左到右水平走向，真正的变道方向为垂直方向（dy）
   修正后横向位移检查能正确反映实际变道行为

③ 短窗口累积角度阈值：10 → 14
   长窗口累积角度阈值：18 → 25
   减少直行车辆（如ID 2）微小角度波动的误报

④ 净角度阈值：6/10 → 10/15
   增强方向主导性要求，减少来回摆动场景的误触发

最终采用的参数：
短窗口8帧、长窗口20帧、ANGLE_STEP=3、异常角度过滤15度、
平滑系数0.4、横向位移窗口15帧、最小横向位移8像素、
横向比例0.08、方向一致性0.50

==================== 五、实验展示 ====================

实验使用2.mp4俯视交通视频进行测试，运行p1_SuspectedLaneChange.py脚本
生成标注视频与CSV分析数据。

[成功检测]
弯道车辆（Track ID 23）：从主路右端向右上弯道行驶，方向角度由约-55°
渐变至-33°，角度变化量约22°。第33帧起三个判定条件（角度变化ok、
净角度ok、横向位移ok）全部通过，被标注为LANE CHANGE。

[未被误报的车辆]
正常直行车辆（Track ID 1, 3, 4, 5, 8, 15）：角度稳定接近0°，
方向一致，未触发误报。
静止车辆（Track ID 6, 7, 9）：被速度过滤拦截，未参与分析。

[漏检情况]
弯道车辆（Track ID 12）：角度有明显变化（-22°至-9°），但横向位移确认
（lateral_ok）不连续，仅在部分帧通过完整判定，存在间歇性漏检。

==================== 六、问题与改进方向 ====================

* 静止/慢速车辆问题：
  虽然加入了速度过滤（min_speed=2.5），但当车辆以极低速度（2-3像素/帧）
  缓慢移动时，仍可能因角度噪声产生误报。改进方案为结合车辆检测框的面积
  变化或光流信息判断真实运动状态。

* 弯道车辆误判问题：
  车辆在弯道上的自然转向与变道行为在角度变化特征上高度相似，单纯依靠
  角度变化难以区分。需要引入车道线信息作为参考基准。

* 远处车辆检测困难：
  俯视画面中远处车辆像素面积小，轨迹点位移量少（vx、vy值小），导致
  角度计算不稳定，且横向位移难以达到阈值。

* 下一阶段改进方向：
  计划引入车道线检测——通过YOLO-seg等语义分割模型提取车道线掩码，
  将车辆中心坐标与车道线进行比对，准确判断车辆是否跨越车道线。
  相比纯角度分析，车道线检测能提供绝对的地理参考，大幅降低弯道等
  场景的误判率，并解决远处车辆像素位移小难以检测的问题。
